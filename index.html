Post #1

1. A felicidade se encontra até nos momentos mais escuros, a se somos capazes de usar bem a luz. 2. São as nossas escolhas as que melhor definem o que somosa, muito mais que nossas habilidades. 3. Não é bom deixar-se levar pelos sonhos e se esquecer de viver. 4. Não tenha pena dos mortos, mas sim dos vivos, ainda mais daqueles que vivem sem amor. 5. Há muitos tipos de valentia. É preciso muita coragem para lutar contra nossos inimigos, mas é preciso o mesmo valor para fazer o mesmo com os amigos. 6. Para uma mente bem preparada, a morte é apenas a próxima grande aventura.
